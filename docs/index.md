# CardSense

CardSense é uma plataforma de Marketing Analytics para segmentação de clientes de cartão de crédito com pipeline de ML não supervisionado.

Principais capacidades:

- Segmentação por clusters com labels de negócio.
- Dashboard executivo com KPIs e qualidade do modelo.
- Exploração do dataset completo com exportação CSV.
- Fluxo de autenticação com usuário demo.
